# News Aggregator API

API для агрегатора новостей из различных источников с аутентификацией JWT.

## Функциональность

- ✅ Регистрация и аутентификация пользователей
- ✅ CRUD операции для источников новостей
- ✅ CRUD операции для статей
- ✅ JWT аутентификация
- ✅ Хеширование паролей (bcrypt)
- ✅ Автоматическая документация API
- ✅ HTML интерфейс
